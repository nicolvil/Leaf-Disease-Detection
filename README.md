
🌿 # Leaf-Disease-Detection
AI-powered plant disease detection system using mobile image recognition — built with Google Teachable Machine &amp; TensorFlow.js

🌿 PlantScan AI — Plant Disease Detector

An AI-based plant disease detection system that allows farmers to capture or 
upload leaf images using their mobile phones and receive instant diagnosis results.

Built as a midterm project for CS Elective 3 at the **University of Northern 
Philippines**, this system aims to provide an accessible, real-time tool for 
early plant disease detection — no expert knowledge or special equipment required.

---

Features

- 📷 Upload or capture leaf images directly from a mobile phone
- 🤖 AI-powered image classification using Google Teachable Machine
- 🌱 Detects 3 conditions: **Healthy**, **Leaf Spot**, and **Blight**
- 📊 Shows confidence percentage and probability breakdown
- 💡 Provides actionable recommendations for each diagnosis
- 📱 Fully mobile-compatible — works in any browser, no app install needed

---

Technologies Used

| Tool | Purpose |
|---|---|
| Google Teachable Machine | Training the image classification model |
| TensorFlow.js | Running the AI model in the browser |
| HTML / CSS / JavaScript | Frontend interface |
| Netlify / GitHub Pages | Deployment |

---

How to Use

1. Open the app link in your browser
2. Choose **Upload Image** or **Use Camera**
3. Select or capture a photo of a plant leaf
4. Click **Analyze Plant**
5. View the diagnosis result and recommended action
